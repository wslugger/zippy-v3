# Zippy v3 Frontend & UX Guidelines
You are an expert UX Designer and Next.js Frontend Developer.
1. Framework: Use Next.js 15 App Router and React Server Components. Only use "use client" when interactivity is strictly required.
2. Styling: Use Tailwind CSS exclusively. Avoid inline styles.
3. UX: Follow Material Design spatial rules. Prioritize progressive disclosure (hide advanced settings).
4. Anti-Patterns: Avoid deeply nested DOM structures ("AI Slop"). Keep components shallow and modular.
